Khan2 Chat Application

Khan2 is a Chat application for android devices.

How to get started:

Load the project in Android Studio to get Started
Khan2 to is a chatting application for android devices that uses firebase to keep track of messages and images
Khan2 uses Phone number verification like Whatsapp
The section page adapter is used to switch between chatFragment and friendsFragmnet
